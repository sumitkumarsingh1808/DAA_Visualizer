# DAA_Visualizer 🚀

An interactive **Design and Analysis of Algorithms (DAA) Visualizer** that helps students and developers understand how different algorithms and data structures work through animations and step-by-step visual representation.  
This project is especially useful for beginners and for interview preparation.

---

## 🧠 Features

- 📊 Visual representation of algorithms
- 🔁 Step-by-step execution
- ⏯ Play, Pause, Reset controls
- ⚡ Speed control for animations
- 📚 Covers important DAA topics:
  - Sorting Algorithms (Bubble, Merge, Quick, etc.)
  - Searching Algorithms (Linear, Binary, etc.)
  - Data Structures (Stack, Queue, Array, etc.)
- 🎯 User-friendly and interactive UI
